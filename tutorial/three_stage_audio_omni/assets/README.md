# Input audio

`input.wav` is the upstream CosyVoice zero-shot prompt clip, copied from:

`https://raw.githubusercontent.com/FunAudioLLM/CosyVoice/main/asset/zero_shot_prompt.wav`

It is included only as a real speech input for the Qwen3-Omni tutorial. Use
`--audio` to replace it with a local recording or a generated WAV.
